# Input
The application is opened in the web browser. No manual input is needed from the user because they are just viewing the preformed resume

# Process
The React application rendered the Resume information using JSX. It organizes the data into sections designed by the coder just as header, education, work experience, and projects. The CSS file is used to format and the style the layout of the web page.

# Output
A formatted resume is displayed as rendered by the application as seen on the browser. The output shows the structured information of the Resume in a pleasant and easily readable format.
